WOODS DYSLEXIA SCREENING TOOL - INSTRUCTOR SETUP
===============================================

LIVE DEMO: https://skewed-lenses.vercel.app

✅ FULLY HOSTED ON VERCEL - No local backend setup required!
✅ AI Analysis (Groq) works directly on Vercel serverless functions

OPTIONAL LOCAL DEVELOPMENT
==========================

Frontend only (AI features work on deployed version):
   cd woods
   npm install
   npm run dev
   # Runs on http://localhost:5173

TECH STACK
==========
Frontend:
- React 19 + Vite 7
- Zustand (State management)
- Web Speech API (Real-time speech recognition)

Backend:
- Vercel Serverless Functions (Node.js)
- Groq AI API (Speech pattern analysis)

Deployment:
- Frontend: Vercel (live)
- Backend: Vercel Serverless Functions (live)

FEATURES IMPLEMENTED
===================
- Real-time speech-to-text with visual feedback
- DIBELS-validated reading assessment
- Word-by-word accuracy tracking
- 3D globe visualization ("1 in 10" dyslexia representation)
- Persistent results storage

TESTING
=======
1. Visit live site: https://skewed-lenses.vercel.app
2. Click "Benchmark 1"
3. Allow microphone permissions
4. Read the displayed words aloud
5. See real-time green/red feedback
6. Complete assessment to see fluency score

Note: All AI analysis features work directly on the live Vercel deployment.
